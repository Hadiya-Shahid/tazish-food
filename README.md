# TAZISH FOOD Business Management System

TAZISH FOOD is a frozen food business
## Goal 
Built a mobile friendly web application that manages our business

The application will help manage orders, generate bills, track expenses, and provide AI-powered insights.

## Core Objectives

- Generate customer bills
- Store all orders automatically
- Sync every order with Google Sheets
- Provide AI-powered business insights
- Track profits, investments, and expenses
- Allow order entry through:
  - Manual typing
  - Voice input
  - Photo recognition

The application will be designed as a web app optimized for mobile devices.

## Platform

Phase 1:
Mobile-friendly Web App (PWA)

Phase 2:
Native mobile apps

## Users

Primary User:
Business Owner (TAZISH FOOD)

Secondary Users:
Customers receiving bills
