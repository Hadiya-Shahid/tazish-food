# Agent Rules (STRICT)

You must follow these rules strictly while building the application.

## Cost Constraint

This project must be built using 100% FREE tools only.

Do NOT use:
- Paid APIs (OpenAI, Claude, etc.)
- Paid SaaS services
- Any tool requiring credit card or billing setup

---

## Allowed Technologies (FREE ONLY)

Frontend:
- Next.js
- TailwindCSS

Backend:
- FastAPI (Python)

Database:
- PostgreSQL via Supabase (free tier)

Hosting:
- Vercel (frontend)
- Render or Railway (backend free tier)

---

## AI Features (FREE IMPLEMENTATION ONLY)

Voice Processing:
- Use Whisper LOCAL (not API)

Image Processing:
- Use Tesseract OCR

AI Parsing:
- Use rule-based parsing (regex, keyword matching)
- Do NOT use LLM APIs

---

## Google Sheets

- Use Google Sheets API (free)
- Use Service Account authentication

---

## Analytics

- Use basic calculations (no paid AI)
- Use Chart.js or similar free library

---

## Code Requirements

- Write clean, modular, production-ready code
- Do not include any paid integrations
- Prefer open-source libraries only

---

## Development Strategy

Build in phases:

1. Order system (manual)
2. Database integration
3. Google Sheets sync
4. Dashboard analytics
5. Voice input
6. Image input

---

## Important

If a feature normally requires a paid service:
→ Replace it with a free/open-source alternative
→ Or implement a simplified version
